# hackaiTUI
**a simple, no bs TUI client for the hackai api**

## instructions
1. get your hackai api key from the dashboard
2. replace `YOUR_API_KEY_GOES_HERE` with your api key
3. run main.py

you can also change the model by replacing `moonshotai/kimi-k2-0905` with any model  
***(hackaiTUI only supports text models, not image models)***
